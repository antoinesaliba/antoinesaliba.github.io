Projects:

Omni Notes

Contributed to the open source Android application Omni Notes with a group of classmates. Spent a total of 5 weeks performing quality assurance tasks such as black and white box testing. Numerous bugs were found and fixed to improve the overall quality and performance of the application.

Inspirator

Built a complete Android application over a four week period utilizing the Twitter API and Google Mobile Ads. Inspirator allows users to set periodic notifications that each has an original motivational message. Users can also tweet what they were motivated to achieve and their tweets show up in the application.

Speed Demon File Manager

Created the front end interface for a file manager using Java Swing. Graphical user interface patterns as well as psychological principles learned from class were incorperated to improve the overal user experience of the application.
